## Features

- Save, Edit, Delete, Add buttons
- Fullscreen mode
- Responsive


## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd todoapp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
